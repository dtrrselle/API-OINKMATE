# 🐾 OinkMate Pet Hub

OinkMate Pet Hub is a simple API-powered website developed as a course project to demonstrate API integration, data retrieval, and data presentation using public APIs. The website allows users to explore dog breed information and discover interesting cat facts through real-time data fetched from the Dog API and Cat Facts API. This project showcases the use of HTML, CSS, JavaScript, Fetch API, error handling, and GitHub collaboration.
